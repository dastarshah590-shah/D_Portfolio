# DASTAR HUSSAIN Portfolio

Professional React portfolio for freelance platforms, client outreach, and custom domain deployment.

## Tech Stack

- React
- Vite
- CSS animations
- Lucide React icons

## Local Development

```bash
npm install
npm run dev
```

On Windows PowerShell, use `npm.cmd` if `npm.ps1` is blocked:

```bash
npm.cmd install
npm.cmd run dev
```

## Production Build

```bash
npm run build
```

The production files will be created in `dist/`.
